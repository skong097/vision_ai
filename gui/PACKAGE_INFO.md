   - 이벤트 자동 로깅
   - 신체 정보 표시

4. **데이터베이스**
   - 사용자 관리
   - 이벤트 로그
   - 자동신고 로그
   - 시스템 설정

## 📊 기술 스택

### Frontend (GUI)
- **PyQt6** - 최신 Qt6 기반 GUI

### AI/ML
- **YOLO Pose** - Keypoint 감지
- **Random Forest** - 낙상 분류
- **OpenCV** - 영상 처리
